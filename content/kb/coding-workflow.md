Agent era allows us to work on multiple work stream simultaneously, but some
things limit us from doing so.

- Too many windows/tabs.
- Git diffs getting mixed.

## Too Many Windows/Tabs

When working with agents, I end up getting lost in tabs. In these tabs, I'm
running these:

- The agent itself. Usually OpenCode or Claude.
- Lazygit for reviewing the diff.
- Code editor, in case I want to check something. Generally Neovim or IntelliJ
- Terminal for running server.
- Terminal for running ad hoc commands.

Sometimes I open a new terminal or use some existing open terminal to do
something that is already running in some other existing terminal.

Now multiply this with the number of task I want to do in parallel. Sometimes it
gets a bit to confusing.

---

At a time, one task runs on a given workspace.

This works well on Niri.
